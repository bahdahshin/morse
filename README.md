# Morse

A minimal iOS app that turns the entire display white while one or more fingers
are touching it, and returns to black when all fingers are lifted.

## Run

1. Open `Morse.xcodeproj` in Xcode.
2. Select an iPhone simulator or a connected iPhone.
3. Press Run.

The app supports both portrait and landscape orientations and hides system
overlays for an uninterrupted full-screen surface.
